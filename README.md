# house-price-prediction

# 🏠 House Price Prediction

A Machine Learning project that predicts house prices based on features like area, number of rooms, and other factors. This project also includes a **Streamlit web app** for user interaction.

---

## 📌 Project Overview

This project uses **Linear Regression** to predict house prices. It demonstrates the full ML pipeline:

* Data preprocessing
* Model training
* Model evaluation
* Deployment using Streamlit

---

## 🚀 Features

* Predict house prices instantly
* Simple and interactive UI using Streamlit
* Trained ML model using real dataset
* Easy to use and deploy

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/house-price-prediction.git
cd house-price-prediction
```

### 2️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run the app

```
streamlit run app.py
```

---

## 📊 How It Works

1. Load dataset using Pandas
2. Select features (area, rooms, etc.)
3. Train model using Linear Regression
4. Save model using pickle
5. Use Streamlit UI for predictions

---

## 📈 Example Input

* Area: 1200 sq.ft
* Rooms: 3
